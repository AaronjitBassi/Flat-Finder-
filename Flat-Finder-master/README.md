# Flat-Finder
## FDM Flat Finder - Software Engineering Group Project [Group 24]

To run the react app:
  - cd client 
  - npm install 
  - npm run build
  - npm start 
  
To run the JSON server: 
  - cd client 
  - npx json-server -p 3500 -w data/db.json
  

Try this, if the cookies are not found: 
- npm install js-cookie
